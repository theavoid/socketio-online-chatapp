## Installation
Dosyaları indirin ve Powershell penceresine `npm i` yazın, bu komut tüm modüllerin yüklenmesini sağlayacaktır. Ardından npm start yazarak aktif edebilirsiniz.
Localhost URL: http://localhost:1337
